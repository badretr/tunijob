💼 TuniJob — Django School Project

I’m pleased to share my school project TuniJob, a web-based recruitment platform connecting job seekers and employers across Tunisia.

🎯 Key Features:
✔️ Candidate / Employer roles with dedicated dashboards
✔️ Complete application workflow and interview management
✔️ Real-time messaging and notifications
✔️ Dynamic job listings with advanced filters
✔️ Analytical dashboards with CSV export
✔️ Modern, responsive user interface

💻 Technologies Used:

Backend: Django

Frontend: HTML5, Tailwind CSS, JavaScript

Database: SQLite

Others: Real-time messaging & notifications, file uploads, testing
